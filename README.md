## Installation

To use this application, you need to install the following modules: mysql2, express, Sequelize, and dotenv

## Usage

To use this app,  clone the repository from GitHub. Using the schema in the db folder, run the sqlcommands in mySQL workbench to create the database. In the terminal, run 'node server.js' to create the table strucctures and 'node seeds/index.js' to seed the created tables.

Open Insomnia core to send a GET, PUT, DELETE, and POST requests from the different routes.
